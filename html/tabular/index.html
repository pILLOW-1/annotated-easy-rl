表格型方法 (Tabular Methods) - Q-Learning & Sarsa

本文件是蘑菇书EasyRL第三章的Python实现。

表格型方法使用表格来存储状态-动作价值函数 Q(s,a) 或状态价值函数 V(s)。 适用于状态空间和动作空间都是离散且较小的场景。

贝尔曼方程

状态价值函数

Vπ(s)=𝔼π[Gt∣st=s]=𝔼π[∑k=0∞γkrt+k+1∣st=s]

贝尔曼期望方程：

Vπ(s)=∑aπ(a|s)∑s′,rp(s′,r|s,a)[r+γVπ(s′)]

动作价值函数

Qπ(s,a)=𝔼π[Gt∣st=s,at=a]=𝔼π[∑k=0∞γkrt+k+1∣st=s,at=a]

贝尔曼最优方程：

Q*(s,a)=∑s′,rp(s′,r|s,a)[r+γmaxa′Q*(s′,a′)]

Q-Learning (异策略)

Q-Learning是异策略(off-policy)的时序差分(TD)控制算法。

Q值更新规则

Q(st,at)←Q(st,at)+α[rt+1+γmaxaQ(st+1,a)−Q(st,at)]

其中： - α 是学习率 - γ 是折扣因子 - rt+1+γmaxaQ(st+1,a) 是TD目标 - rt+1+γmaxaQ(st+1,a)−Q(st,at) 是TD误差

Q-Learning使用 maxaQ(st+1,a) 来估计下一状态的价值， 这对应于贪婪策略 π(s)=\argmaxaQ(s,a)， 但行为策略可以是 ε-贪心策略，因此是异策略的。

Sarsa (同策略)

Sarsa是同策略(on-policy)的时序差分(TD)控制算法。

Q值更新规则

Q(st,at)←Q(st,at)+α[rt+1+γQ(st+1,at+1)−Q(st,at)]

Sarsa使用实际采取的下一个动作 at+1 的Q值来更新， 因此学习的就是行为策略本身的价值，是同策略的。

Q-Learning vs Sarsa

特性 Q-Learning Sarsa
策略类型 异策略(off-policy) 同策略(on-policy)
TD目标 r+γmaxaQ(s′,a) r+γQ(s′,a′)
行为策略 ε-贪心 ε-贪心
目标策略 贪婪 ε-贪心
收敛性 收敛到最优Q值 收敛到行为策略的Q值

价值迭代 (Value Iteration)

价值迭代是一种动态规划算法，用于求解已知环境模型的MDP。

更新规则

Vk+1(s)=maxa∑s′,rp(s′,r|s,a)[r+γVk(s′)]

价值迭代直接更新状态价值函数，直到收敛。 收敛后，最优策略可以通过贪婪选择得到：

π*(s)=\argmaxa∑s′,rp(s′,r|s,a)[r+γV*(s′)]

-贪心策略

π(a|s)={1−ε+ε|𝒜|if a=\argmaxa′Q(s,a′)ε|𝒜|otherwise

或者简化为：

a={\argmaxaQ(s,a)概率 1−ε随机动作概率 ε

Q-Learning智能体

Q-Learning更新规则：

Q(s,a)←Q(s,a)+α[r+γmaxa′Q(s′,a′)−Q(s,a)]

其中： - α：学习率 - γ：折扣因子 - r+γmaxa′Q(s′,a′)：TD目标 - r+γmaxa′Q(s′,a′)−Q(s,a)：TD误差 δ

使用ε-贪心策略选择动作。

a={\argmaxaQ(s,a)概率 1−ε随机动作概率 ε

更新Q值。

Q(s,a)←Q(s,a)+α[r+γmaxa′Q(s′,a′)×(1−done)−Q(s,a)]

参数： - state: st — 当前状态 - action: at — 当前动作 - reward: rt+1 — 奖励 - next_state: st+1 — 下一状态 - done: 是否episode结束

Sarsa智能体

Sarsa更新规则：

Q(s,a)←Q(s,a)+α[r+γQ(s′,a′)−Q(s,a)]

与Q-Learning不同，Sarsa使用实际采取的下一个动作 a′ 的Q值， 因此是同策略(on-policy)算法。

ε-贪心策略选择动作。

更新Q值。

Q(s,a)←Q(s,a)+α[r+γQ(s′,a′)×(1−done)−Q(s,a)]

注意：Sarsa需要下一个动作 a′ 来更新，这是同策略的关键。

价值迭代智能体

价值迭代是一种动态规划算法，适用于已知环境模型的场景。

更新规则：

Vk+1(s)=maxa∑s′,rp(s′,r|s,a)[r+γVk(s′)]

收敛后，最优策略：

π*(s)=\argmaxa∑s′,rp(s′,r|s,a)[r+γV*(s′)]

设置环境模型。

参数： - P: 状态转移模型 P[s][a] = [(prob, next_state, reward, done), ...] 其中prob是转移到next_state的概率

执行一次价值迭代。

Vk+1(s)=maxa∑s′,rp(s′,r|s,a)[r+γVk(s′)]

返回值： - 价值函数的最大变化量

求解最优价值函数和策略。

迭代直到价值函数收敛（变化小于theta）或达到最大迭代次数。

从最优价值函数提取最优策略。

π*(s)=\argmaxa∑s′,rp(s′,r|s,a)[r+γV*(s′)]

2
3    def __init__(self, state_dim: int, action_dim: int, gamma: float = 0.99, theta: float = 1e-6):
4        self.state_dim = state_dim
5        self.action_dim = action_dim
6        self.gamma = gamma
7        self.theta = theta  # 收敛阈值
8        self.value_table = np.zeros(state_dim, dtype=np.float32)
9        self.policy = np.zeros(state_dim, dtype=np.int32)
10 # 环境模型：P[s][a] = [(prob, next_state, reward, done), ...]
11        self.P = None
12
13    def set_model(self, P: dict):
14        self.P = P
15
16    def iterate(self):
17        delta = 0
18        for s in range(self.state_dim):
19            v = self.value_table[s]
20 # 计算每个动作的期望价值
21            action_values = np.zeros(self.action_dim)
22            for a in range(self.action_dim):
23                for prob, next_state, reward, done in self.P[s][a]:
24 # 期望价值：
25                    action_values[a] += prob * (reward + self.gamma * self.value_table[next_state] * (1 - done))
26
27 # 选择最大价值：
28            self.value_table[s] = np.max(action_values)
29            delta = max(delta, abs(v - self.value_table[s]))
30
31        return delta
32
33    def solve(self, max_iterations: int = 1000):
34        for _ in range(max_iterations):
35            delta = self.iterate()
36            if delta &lt; self.theta:
37                break
38
39 # 提取最优策略
40        self.extract_policy()
41
42    def extract_policy(self):
43        for s in range(self.state_dim):
44            action_values = np.zeros(self.action_dim)
45            for a in range(self.action_dim):
46                for prob, next_state, reward, done in self.P[s][a]:
47                    action_values[a] += prob * (reward + self.gamma * self.value_table[next_state] * (1 - done))
48            self.policy[s] = np.argmax(action_values)
