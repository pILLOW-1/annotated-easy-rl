9# 表格型方法 (Tabular Methods) - Q-Learning & Sarsa
10
11本文件是[蘑菇书EasyRL](https://github.com/datawhalechina/easy-rl/)第三章的Python实现。
12
13表格型方法使用表格来存储状态-动作价值函数 Q(s, a) 或状态价值函数 V(s)。
14适用于状态空间和动作空间都是离散且较小的场景。
15
16## 贝尔曼方程
17
18### 状态价值函数
19
20V^\pi(s) = \mathbb{E}_\pi\left[G_t \mid s_t = s\right] = \mathbb{E}_\pi\left[\sum_{k=0}^{\infty} \gamma^k r_{t+k+1} \mid s_t = s\right]
21
22贝尔曼期望方程：
23V^\pi(s) = \sum_a \pi(a|s) \sum_{s', r} p(s', r|s, a) \left[r + \gamma V^\pi(s')\right]
24
25### 动作价值函数
26
27Q^\pi(s, a) = \mathbb{E}_\pi\left[G_t \mid s_t = s, a_t = a\right] = \mathbb{E}_\pi\left[\sum_{k=0}^{\infty} \gamma^k r_{t+k+1} \mid s_t = s, a_t = a\right]
28
29贝尔曼最优方程：
30Q^*(s, a) = \sum_{s', r} p(s', r|s, a) \left[r + \gamma \max_{a'} Q^*(s', a')\right]
31
32## Q-Learning (异策略)
33
34Q-Learning是异策略(off-policy)的时序差分(TD)控制算法。
35
36### Q值更新规则
37
38Q(s_t, a_t) \leftarrow Q(s_t, a_t) + \alpha \left[r_{t+1} + \gamma \max_a Q(s_{t+1}, a) - Q(s_t, a_t)\right]
39
40其中：
41- \alpha 是学习率
42- \gamma 是折扣因子
43- r_{t+1} + \gamma \max_a Q(s_{t+1}, a) 是TD目标
44- r_{t+1} + \gamma \max_a Q(s_{t+1}, a) - Q(s_t, a_t) 是TD误差
45
46Q-Learning使用 \max_a Q(s_{t+1}, a) 来估计下一状态的价值，
47这对应于贪婪策略 \pi(s) = \arg\max_a Q(s, a)，
48但行为策略可以是 \varepsilon-贪心策略，因此是异策略的。
49
50## Sarsa (同策略)
51
52Sarsa是同策略(on-policy)的时序差分(TD)控制算法。
53
54### Q值更新规则
55
56Q(s_t, a_t) \leftarrow Q(s_t, a_t) + \alpha \left[r_{t+1} + \gamma Q(s_{t+1}, a_{t+1}) - Q(s_t, a_t)\right]
57
58Sarsa使用实际采取的下一个动作 a_{t+1} 的Q值来更新，
59因此学习的就是行为策略本身的价值，是同策略的。
60
61## Q-Learning vs Sarsa
62
63| 特性 | Q-Learning | Sarsa |
64|------|-----------|-------|
65| 策略类型 | 异策略(off-policy) | 同策略(on-policy) |
66| TD目标 | r + \gamma \max_a Q(s', a) | r + \gamma Q(s', a') |
67| 行为策略 | \varepsilon-贪心 | \varepsilon-贪心 |
68| 目标策略 | 贪婪 | \varepsilon-贪心 |
69| 收敛性 | 收敛到最优Q值 | 收敛到行为策略的Q值 |
70
71## 价值迭代 (Value Iteration)
72
73价值迭代是一种动态规划算法，用于求解已知环境模型的MDP。
74
75### 更新规则
76
77V_{k+1}(s) = \max_a \sum_{s', r} p(s', r|s, a) \left[r + \gamma V_k(s')\right]
78
79价值迭代直接更新状态价值函数，直到收敛。
80收敛后，最优策略可以通过贪婪选择得到：
81
82\pi^*(s) = \arg\max_a \sum_{s', r} p(s', r|s, a) \left[r + \gamma V^*(s')\right]
83
84## \varepsilon-贪心策略
85
86\pi(a|s) = \begin{cases}
871 - \varepsilon + \frac{\varepsilon}{|\mathcal{A}|} & \text{if } a = \arg\max_{a'} Q(s, a') \\
88\frac{\varepsilon}{|\mathcal{A}|} & \text{otherwise}
89\end{cases}
90
91或者简化为：
92a = \begin{cases}
93\arg\max_a Q(s, a) & \text{概率 } 1-\varepsilon \\
94\text{随机动作} & \text{概率 } \varepsilon
95\end{cases}

def init(self, state_dim: int, action_dim: int, gamma: float = 0.99, theta: float = 1e-6): self.state_dim = state_dim self.action_dim = action_dim self.gamma = gamma self.theta = theta # 收敛阈值 self.value_table = np.zeros(state_dim, dtype=np.float32) self.policy = np.zeros(state_dim, dtype=np.int32) # 环境模型：P[s][a] = [(prob, next_state, reward, done), ...] self.P = None

def set_model(self, P: dict):

self.P = P

def iterate(self): r

delta = 0 for s in range(self.state_dim): v = self.value_table[s] # 计算每个动作的期望价值 action_values = np.zeros(self.action_dim) for a in range(self.action_dim): for prob, next_state, reward, done in self.P[s][a]: # 期望价值：∑s′,rp(s′,r|s,a)[r+γV(s′)] action_values[a] += prob * (reward + self.gamma * self.value_table[next_state] * (1 - done))

    # 选择最大价值：<span class="math-inline"><math xmlns="http://www.w3.org/1998/Math/MathML" display="inline"><mrow><msub><mi>V</mi><mrow><mi>k</mi><mo>&#x0002B;</mo><mn>1</mn></mrow></msub><mo stretchy="false">&#x00028;</mo><mi>s</mi><mo stretchy="false">&#x00029;</mo><mo>&#x0003D;</mo><msub><mo>max</mo><mi>a</mi></msub><mi>Q</mi><mo stretchy="false">&#x00028;</mo><mi>s</mi><mo>&#x0002C;</mo><mi>a</mi><mo stretchy="false">&#x00029;</mo></mrow></math></span>
    self.value_table[s] = np.max(action_values)
    delta = max(delta, abs(v - self.value_table[s]))

return delta

def solve(self, max_iterations: int = 1000):

for _ in range(max_iterations): delta = self.iterate() if delta < self.theta: break

# 提取最优策略
self.extract_policy()

def extract_policy(self): r

for s in range(self.state_dim): action_values = np.zeros(self.action_dim) for a in range(self.action_dim): for prob, next_state, reward, done in self.P[s][a]: action_values[a] += prob * (reward + self.gamma * self.value_table[next_state] * (1 - done)) self.policy[s] = np.argmax(action_values)
