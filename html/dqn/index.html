Q函数与贝尔曼方程

在强化学习中，Q函数定义为：

Qπ(s,a)=𝔼π[∑k=0∞γkrt+k+1∣st=s,at=a]

最优Q函数满足贝尔曼最优方程：

Q*(s,a)=𝔼s′~ℰ[r+γmaxa′Q*(s′,a′)∣s,a]

DQN的核心思想

DQN使用神经网络来近似Q函数：Q(s,a;θ)≈Q*(s,a)

训练目标是使Q函数的预测满足贝尔曼方程。定义目标值：

yi=ri+γmaxa′Q(si+1,a′;θ−)

其中 θ− 是目标网络的参数，定期从主网络 θ 复制。

损失函数为：

L(θ)=𝔼(s,a,r,s′)~U(D)[(yi−Q(s,a;θ))2]

目标网络 🎯

为了提高训练稳定性，DQN使用两个网络： - 主网络 (policy network)：参数 θ，用于选择动作和计算Q值 - 目标网络 (target network)：参数 θ−，定期从主网络复制，用于计算目标值

目标值：

yi=ri+γmaxa′Q(si+1,a′;θ−)

目标网络的参数 θ− 每隔一定步数更新为 θ，这打破了样本间的相关性，提高了训练稳定性。

Double DQN

标准DQN中，max操作符使用同一个网络来选择和评估动作：

maxa′Q(s′,a′;θ)=Q(s′,\argmaxa′Q(s′,a′;θ);θ)

这会导致Q值的高估。Double DQN解耦了动作选择和价值评估：

maxa′Q(s′,a′;θ−)=Q(s′,\argmaxa′Q(s′,a′;θ);θ−)
  • 使用主网络 θ 选择动作：\argmaxa′Q(s′,a′;θ)
  • 使用目标网络 θ− 评估价值：Q(s′,best_action;θ−)

Double DQN的损失函数：

L(θ)=𝔼[(r+γQ(s′,\argmaxa′Q(s′,a′;θ);θ−)−Q(s,a;θ))2]

Dueling Network ⚔️

Dueling网络将Q值分解为状态价值和优势函数：

Qπ(s,a)&=Vπ(s)+Aπ(s,a)𝔼a~π(s)[Aπ(s,a)]&=0

实际计算中：

Q(s,a)=V(s)+(A(s,a)−1|𝒜|∑a′∈𝒜A(s,a′))

Dueling网络的优势在于： - 在很多状态下，动作并不重要（优势函数接近0） - 在少数关键状态下，动作非常重要 - 这种结构让网络可以更好地学习这种差异

经验回放

DQN使用经验回放缓冲区来存储和采样经验：

D={(st,at,rt,st+1)}

训练时从缓冲区中随机采样：

(s,a,r,s′)~U(D)

经验回放的好处： 1. 打破样本间的时间相关性 2. 提高数据利用率 3. 使训练更加稳定

探索策略

ε-贪心策略

a={\argmaxaQ(s,a)概率 1−ε随机动作概率 ε

ε 通常从大到小衰减，初期多探索，后期多利用。

Boltzmann探索

π(a|s)=eQ(s,a)/T∑a′eQ(s,a′)/T

其中 T 是温度参数，控制探索程度。

目标网络 🎯

使用经验回放 U(D) 随机采样历史经验，并使用独立的Q网络 参数 \textcolororangeθ− 来计算目标值。 \textcolororangeθ− 定期更新。

损失函数：

ℒ(θ)=𝔼(s,a,r,s′)~U(D)[(r+γmaxa′Q(s′,a′;\textcolororangeθ−)−Q(s,a;θ))2]

Double Q-Learning

使用Double Q-Learning，其中： - \textcolorcyanθ 用于选择动作 - \textcolororangeθ− 用于评估价值

ℒ(θ)=𝔼[(&r+γ\textcolororangeQ(s′,\argmaxa′\textcolorcyanQ(s′,a′;\textcolorcyanθ);\textcolororangeθ−)&−Q(s,a;θ))2]

计算DQN损失。

参数映射： - q: Q(s,a;θ) — 主网络输出的Q值 - action: a — 采取的动作 - double_q: \textcolorcyanQ(s′;\textcolorcyanθ) — 主网络对下一状态的Q值 - target_q: \textcolororangeQ(s′;\textcolororangeθ−) — 目标网络对下一状态的Q值 - done: 是否episode结束 - reward: r — 奖励 - weights: 优先级经验回放的权重

返回值： - TD误差（用于优先级回放） - 损失值

1class QFuncLoss(nn.Module):
2
3    def __init__(self, gamma: float):
4        super().__init__()
5        self.gamma = gamma
6        self.huber_loss = nn.SmoothL1Loss(reduction="none")
7
8    def forward(
9        self,
10        q: torch.Tensor,
11        action: torch.Tensor,
12        double_q: torch.Tensor,
13        target_q: torch.Tensor,
14        done: torch.Tensor,
15        reward: torch.Tensor,
16        weights: torch.Tensor = None,
17    ) -&gt; Tuple[torch.Tensor, torch.Tensor]:
18        # Q(s,a;\theta) — 选择当前状态-动作对的Q值
19        q_sampled_action = q.gather(-1, action.to(torch.long).unsqueeze(-1)).squeeze(-1)
20
21        # 不传播梯度到目标网络
22        # r + \gamma \textcolor{orange}{Q}\Big(s', \underset{a'}{\arg\max} \textcolor{cyan}{Q}(s', a'; \textcolor{cyan}{\theta}); \textcolor{orange}{\theta^-}\Big)
23        with torch.no_grad():
24            # 使用主网络选择最佳动作
25            # \underset{a'}{\arg\max} \textcolor{cyan}{Q}(s', a'; \textcolor{cyan}{\theta})
26            best_next_action = torch.argmax(double_q, -1)
27
28            # 使用目标网络评估最佳动作的价值
29            # \textcolor{orange}{Q}\Big(s', \underset{a'}{\arg\max} \textcolor{cyan}{Q}(s', a'; \textcolor{cyan}{\theta}); \textcolor{orange}{\theta^-}\Big)
30            best_next_q_value = target_q.gather(-1, best_next_action.unsqueeze(-1)).squeeze(-1)
31
32            # 计算目标Q值
33            # 乘以 `(1 - done)` 确保episode结束时不加上下一状态的价值
34            # r + \gamma \textcolor{orange}{Q}\Big(s', \text{best\_action}; \textcolor{orange}{\theta^-}\Big)
35            q_update = reward + self.gamma * best_next_q_value * (1 - done)
36
37            # TD误差 \delta = Q(s,a;\theta) - y
38            td_error = q_sampled_action - q_update
39
40        # 使用Huber损失代替MSE，对异常值更不敏感
41        losses = self.huber_loss(q_sampled_action, q_update)
42
43        if weights is not None:
44            # 优先级经验回放的加权损失
45            loss = torch.mean(weights * losses)
46        else:
47            loss = losses.mean()
48
49        return td_error, loss

Dueling网络 ⚔️ 结构

使用Dueling网络来计算Q值。

Dueling网络的直觉： - 在大多数状态下，动作并不重要 - 在少数状态下，动作非常关键 - Dueling网络可以很好地表示这种差异

Qπ(s,a)&=Vπ(s)+Aπ(s,a)𝔼a~π(s)[Aπ(s,a)]&=0

实际计算：

Q(s,a)=V(s)+(A(s,a)−1|𝒜|∑a′∈𝒜A(s,a′))

V 和 A 网络共享前面的卷积层。

1class DuelingQNetwork(nn.Module):
2
3    def __init__(self, state_dim: int, action_dim: int, hidden_dim: int = 256):
4        super().__init__()
5        # 共享特征提取器
6        self.feature = nn.Sequential(
7            nn.Linear(state_dim, hidden_dim),
8            nn.ReLU(),
9            nn.Linear(hidden_dim, hidden_dim),
10            nn.ReLU(),
11        )
12
13        # 状态价值头 V(s)
14        self.state_value = nn.Sequential(
15            nn.Linear(hidden_dim, hidden_dim // 2),
16            nn.ReLU(),
17            nn.Linear(hidden_dim // 2, 1),
18        )
19
20        # 优势函数头 A(s, a)
21        self.action_value = nn.Sequential(
22            nn.Linear(hidden_dim, hidden_dim // 2),
23            nn.ReLU(),
24            nn.Linear(hidden_dim // 2, action_dim),
25        )
26
27    def forward(self, state: torch.Tensor) -&gt; torch.Tensor:
28        # 共享特征提取
29        features = self.feature(state)
30
31        # V(s) — 状态价值
32        state_value = self.state_value(features)
33
34        # A(s, a) — 动作优势
35        action_value = self.action_value(features)
36
37        # A(s, a) - \frac{1}{|\mathcal{A}|}\sum_{a'} A(s, a') — 中心化的优势函数
38        action_score_centered = action_value - action_value.mean(dim=-1, keepdim=True)
39
40        # Q(s, a) = V(s) + \left(A(s, a) - \frac{1}{|\mathcal{A}|}\sum_{a'} A(s, a')\right)
41        q = state_value + action_score_centered
42
43        return q

ε-贪心探索策略

a={\argmaxaQ(s,a)概率 1−ε随机动作概率 ε

ε 随时间衰减：

εt=εend+(εstart−εend)×e−t/decay

根据ε-贪心策略选择动作。

参数： - q_values: Q(s,a;θ) — 当前状态的Q值

返回值： - 选择的动作

1class EpsilonGreedy:
2
3    def __init__(
4        self,
5        action_dim: int,
6        epsilon_start: float = 1.0,
7        epsilon_end: float = 0.01,
8        decay: int = 1000,
9    ):
10        self.action_dim = action_dim
11        self.epsilon_start = epsilon_start
12        self.epsilon_end = epsilon_end
13        self.decay = decay
14        self.epsilon = epsilon_start
15        self.count = 0
16
17    def select_action(self, q_values: torch.Tensor) -&gt; int:
18        self.count += 1
19        # 衰减\varepsilon
20        self.epsilon = self.epsilon_end + (self.epsilon_start - self.epsilon_end) * (
21            -self.count / self.decay
22        )
23        self.epsilon = max(self.epsilon, self.epsilon_end)
24
25        if torch.rand(1).item() &lt; self.epsilon:
26            # 探索：随机动作
27            return torch.randint(0, self.action_dim, (1,)).item()
28        else:
29            # 利用：选择Q值最大的动作
30            return torch.argmax(q_values).item()
