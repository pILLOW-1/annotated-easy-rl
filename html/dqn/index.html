9# 深度Q网络 (DQN) - Deep Q Networks
10
11本文件是[蘑菇书EasyRL](https://github.com/datawhalechina/easy-rl/)第六章的PyTorch实现，
12参考论文[Playing Atari with Deep Reinforcement Learning](https://arxiv.org/abs/1312.5602)，
13以及[Dueling Network](https://arxiv.org/abs/1511.06581)和
14[Double Q-Learning](https://arxiv.org/abs/1509.06461)。
15
16## Q函数与贝尔曼方程
17
18在强化学习中，Q函数定义为：
19
20Q^\pi(s, a) = \mathbb{E}_\pi\left[\sum_{k=0}^{\infty} \gamma^k r_{t+k+1} \mid s_t=s, a_t=a\right]
21
22最优Q函数满足贝尔曼最优方程：
23
24Q^*(s, a) = \mathbb{E}_{s' \sim \mathcal{E}}\left[r + \gamma \max_{a'} Q^*(s', a') \mid s, a\right]
25
26## DQN的核心思想
27
28DQN使用神经网络来近似Q函数：Q(s, a; \theta) \approx Q^*(s, a)
29
30训练目标是使Q函数的预测满足贝尔曼方程。定义目标值：
31
32y_i = r_i + \gamma \max_{a'} Q(s_{i+1}, a'; \theta^-)
33
34其中 \theta^- 是目标网络的参数，定期从主网络 \theta 复制。
35
36损失函数为：
37
38L(\theta) = \mathbb{E}_{(s,a,r,s') \sim U(D)}\left[(y_i - Q(s, a; \theta))^2\right]
39
40## 目标网络 🎯
41
42为了提高训练稳定性，DQN使用两个网络：
43- **主网络** (policy network)：参数 \theta，用于选择动作和计算Q值
44- **目标网络** (target network)：参数 \theta^-，定期从主网络复制，用于计算目标值
45
46目标值：
47y_i = r_i + \gamma \max_{a'} Q(s_{i+1}, a'; \theta^-)
48
49目标网络的参数 \theta^- 每隔一定步数更新为 \theta，这打破了样本间的相关性，提高了训练稳定性。
50
51## Double DQN
52
53标准DQN中，max操作符使用同一个网络来选择和评估动作：
54
55\max_{a'} Q(s', a'; \theta) = Q\left(s', \underset{a'}{\arg\max} Q(s', a'; \theta); \theta\right)
56
57这会导致Q值的高估。Double DQN解耦了动作选择和价值评估：
58
59\max_{a'} Q(s', a'; \theta^-) = Q\left(s', \underset{a'}{\arg\max} Q(s', a'; \theta); \theta^-\right)
60
61- 使用主网络 \theta 选择动作：\underset{a'}{\arg\max} Q(s', a'; \theta)
62- 使用目标网络 \theta^- 评估价值：Q(s', \text{best\_action}; \theta^-)
63
64Double DQN的损失函数：
65
66L(\theta) = \mathbb{E}\left[\left(r + \gamma Q\left(s', \underset{a'}{\arg\max} Q(s', a'; \theta); \theta^-\right) - Q(s, a; \theta)\right)^2\right]
67
68## Dueling Network ⚔️
69
70Dueling网络将Q值分解为状态价值和优势函数：
71
72\begin{align}
73Q^\pi(s, a) &= V^\pi(s) + A^\pi(s, a) \\
74\mathbb{E}_{a \sim \pi(s)}\left[A^\pi(s, a)\right] &= 0
75\end{align}
76
77实际计算中：
78
79Q(s, a) = V(s) + \left(A(s, a) - \frac{1}{|\mathcal{A}|}\sum_{a' \in \mathcal{A}} A(s, a')\right)
80
81Dueling网络的优势在于：
82- 在很多状态下，动作并不重要（优势函数接近0）
83- 在少数关键状态下，动作非常重要
84- 这种结构让网络可以更好地学习这种差异
85
86## 经验回放
87
88DQN使用经验回放缓冲区来存储和采样经验：
89
90D = \{(s_t, a_t, r_t, s_{t+1})\}
91
92训练时从缓冲区中随机采样：
93(s, a, r, s') \sim U(D)
94
95经验回放的好处：
961. 打破样本间的时间相关性
972. 提高数据利用率
983. 使训练更加稳定
99
100## 探索策略
101
102### \varepsilon-贪心策略
103
104a = \begin{cases}
105\underset{a}{\arg\max} Q(s, a) & \text{概率 } 1-\varepsilon \\
106\text{随机动作} & \text{概率 } \varepsilon
107\end{cases}
108
109\varepsilon 通常从大到小衰减，初期多探索，后期多利用。
110
111### Boltzmann探索
112
113\pi(a|s) = \frac{e^{Q(s,a)/T}}{\sum_{a'} e^{Q(s,a')/T}}
114
115其中 T 是温度参数，控制探索程度。
116"""

38

40        self.action_value = nn.Sequential(

def init( self, action_dim: int, epsilon_start: float = 1.0, epsilon_end: float = 0.01, decay: int = 1000, ): self.action_dim = action_dim self.epsilon_start = epsilon_start self.epsilon_end = epsilon_end self.decay = decay self.epsilon = epsilon_start self.count = 0

def select_action(self, q_values: torch.Tensor) -> int: r

self.count += 1

衰减ε

self.epsilon = self.epsilon_end + (self.epsilon_start - self.epsilon_end) * ( -self.count / self.decay ) self.epsilon = max(self.epsilon, self.epsilon_end)

if torch.rand(1).item() < self.epsilon: # 探索：随机动作 return torch.randint(0, self.action_dim, (1,)).item() else: # 利用：选择Q值最大的动作 return torch.argmax(q_values).item()

1class EpsilonGreedy:
2    r"""
3    ## \varepsilon-贪心探索策略
4
5    a = \begin{cases}
6    \underset{a}{\arg\max} Q(s, a) & \text{概率 } 1-\varepsilon \\
7    \text{随机动作} & \text{概率 } \varepsilon
8    \end{cases}
9
10    \varepsilon 随时间衰减：
11    \varepsilon_t = \varepsilon_{end} + (\varepsilon_{start} - \varepsilon_{end}) \times e^{-t/\text{decay}}
12        根据\varepsilon-贪心策略选择动作。
13
14        参数：
15        - `q_values`: Q(s, a;\theta) — 当前状态的Q值
16
17        返回值：
18        - 选择的动作
19        self.count += 1
20        # 衰减\varepsilon
21        self.epsilon = self.epsilon_end + (self.epsilon_start - self.epsilon_end) * (
22            -self.count / self.decay
23        )
24        self.epsilon = max(self.epsilon, self.epsilon_end)
25
26        if torch.rand(1).item() < self.epsilon:
27            # 探索：随机动作
28            return torch.randint(0, self.action_dim, (1,)).item()
