Q函数与贝尔曼方程

在强化学习中，Q函数定义为：

$$Q^\pi(s, a) = \mathbb{E}_\pi\left[\sum_{k=0}^{\infty} \gamma^k r_{t+k+1} \mid s_t=s, a_t=a\right]$$

最优Q函数满足贝尔曼最优方程：

$$Q^*(s, a) = \mathbb{E}_{s' \sim \mathcal{E}}\left[r + \gamma \max_{a'} Q^*(s', a') \mid s, a\right]$$

DQN的核心思想

DQN使用神经网络来近似Q函数：$Q(s, a; \theta) \approx Q^*(s, a)$

训练目标是使Q函数的预测满足贝尔曼方程。定义目标值：

$$y_i = r_i + \gamma \max_{a'} Q(s_{i+1}, a'; \theta^-)$$

其中 $\theta^-$ 是目标网络的参数，定期从主网络 $\theta$ 复制。

损失函数为：

$$L(\theta) = \mathbb{E}_{(s,a,r,s') \sim U(D)}\left[(y_i - Q(s, a; \theta))^2\right]$$

目标网络 🎯

为了提高训练稳定性，DQN使用两个网络： - 主网络 (policy network)：参数 $\theta$，用于选择动作和计算Q值 - 目标网络 (target network)：参数 $\theta^-$，定期从主网络复制，用于计算目标值

目标值：

$$y_i = r_i + \gamma \max_{a'} Q(s_{i+1}, a'; \theta^-)$$

目标网络的参数 $\theta^-$ 每隔一定步数更新为 $\theta$，这打破了样本间的相关性，提高了训练稳定性。

Double DQN

标准DQN中，max操作符使用同一个网络来选择和评估动作：

$$\max_{a'} Q(s', a'; \theta) = Q\left(s', \underset{a'}{\arg\max} Q(s', a'; \theta); \theta\right)$$

这会导致Q值的高估。Double DQN解耦了动作选择和价值评估：

$$\max_{a'} Q(s', a'; \theta^-) = Q\left(s', \underset{a'}{\arg\max} Q(s', a'; \theta); \theta^-\right)$$
  • 使用主网络 $\theta$ 选择动作：$\underset{a'}{\arg\max} Q(s', a'; \theta)$
  • 使用目标网络 $\theta^-$ 评估价值：$Q(s', \text{best\_action}; \theta^-)$

Double DQN的损失函数：

$$L(\theta) = \mathbb{E}\left[\left(r + \gamma Q\left(s', \underset{a'}{\arg\max} Q(s', a'; \theta); \theta^-\right) - Q(s, a; \theta)\right)^2\right]$$

Dueling Network ⚔️

Dueling网络将Q值分解为状态价值和优势函数：

$$\begin{align} Q^\pi(s, a) &= V^\pi(s) + A^\pi(s, a) \\ \mathbb{E}_{a \sim \pi(s)}\left[A^\pi(s, a)\right] &= 0 \end{align}$$

实际计算中：

$$Q(s, a) = V(s) + \left(A(s, a) - \frac{1}{|\mathcal{A}|}\sum_{a' \in \mathcal{A}} A(s, a')\right)$$

Dueling网络的优势在于： - 在很多状态下，动作并不重要（优势函数接近0） - 在少数关键状态下，动作非常重要 - 这种结构让网络可以更好地学习这种差异

经验回放

DQN使用经验回放缓冲区来存储和采样经验：

$$D = \{(s_t, a_t, r_t, s_{t+1})\}$$

训练时从缓冲区中随机采样：

$$(s, a, r, s') \sim U(D)$$

经验回放的好处： 1. 打破样本间的时间相关性 2. 提高数据利用率 3. 使训练更加稳定

探索策略

$\varepsilon$-贪心策略

$$a = \begin{cases} \underset{a}{\arg\max} Q(s, a) & \text{概率 } 1-\varepsilon \\ \text{随机动作} & \text{概率 } \varepsilon \end{cases}$$ $\varepsilon$

通常从大到小衰减，初期多探索，后期多利用。

Boltzmann探索

$$\pi(a|s) = \frac{e^{Q(s,a)/T}}{\sum_{a'} e^{Q(s,a')/T}}$$

其中 $T$ 是温度参数，控制探索程度。

DQN损失函数

我们想要找到最优的动作价值函数：

$$\begin{align} Q^*(s,a) &= \max_\pi \mathbb{E}\left[r_t + \gamma r_{t+1} + \gamma^2 r_{t+2} + \cdots \mid s_t=s, a_t=a, \pi\right] \\ Q^*(s,a) &= \mathbb{E}_{s' \sim \mathcal{E}}\left[r + \gamma \max_{a'} Q^*(s', a') \mid s, a\right] \end{align}$$

目标网络 🎯

使用经验回放 $U(D)$ 随机采样历史经验，并使用独立的Q网络 参数 $\textcolor{orange}{\theta^-}$ 来计算目标值。

$\textcolor{orange}{\theta^-}$

定期更新。

损失函数：

$$\mathcal{L}(\theta) = \mathbb{E}_{(s,a,r,s') \sim U(D)}\left[\left(r + \gamma \max_{a'} Q(s', a'; \textcolor{orange}{\theta^-}) - Q(s,a;\theta)\right)^2\right]$$

Double Q-Learning

使用Double Q-Learning，其中： - $\textcolor{cyan}{\theta}$ 用于选择动作 - $\textcolor{orange}{\theta^-}$ 用于评估价值

$$\begin{align} \mathcal{L}(\theta) = \mathbb{E}\Bigg[\bigg(&r + \gamma \textcolor{orange}{Q}\Big(s', \underset{a'}{\arg\max} \textcolor{cyan}{Q}(s', a'; \textcolor{cyan}{\theta}); \textcolor{orange}{\theta^-}\Big) \\ &- Q(s,a;\theta)\bigg)^2\Bigg] \end{align}$$

计算DQN损失。

参数映射： - q: $Q(s,a;\theta)$ — 主网络输出的Q值 - action: $a$ — 采取的动作 - double_q: $\textcolor{cyan}{Q}(s';\textcolor{cyan}{\theta})$ — 主网络对下一状态的Q值 - target_q: $\textcolor{orange}{Q}(s';\textcolor{orange}{\theta^-})$ — 目标网络对下一状态的Q值 - done: 是否episode结束 - reward: $r$ — 奖励 - weights: 优先级经验回放的权重

返回值： - TD误差（用于优先级回放） - 损失值

32            # 计算目标Q值

34            # $$r + \gamma \textcolor{orange}{Q}\Big(s', \text{best\_action}; \textcolor{orange}{\theta^-}\Big)$$
35            q_update = reward + self.gamma * best_next_q_value * (1 - done)
36
37            # TD误差 $\delta = Q(s,a;\theta) - y$
38            td_error = q_sampled_action - q_update
39
40        # 使用Huber损失代替MSE，对异常值更不敏感
41        losses = self.huber_loss(q_sampled_action, q_update)
42
43        if weights is not None:

45            loss = torch.mean(weights * losses)

Dueling网络 ⚔️ 结构

使用Dueling网络来计算Q值。

Dueling网络的直觉： - 在大多数状态下，动作并不重要 - 在少数状态下，动作非常关键 - Dueling网络可以很好地表示这种差异

$$\begin{align} Q^\pi(s,a) &= V^\pi(s) + A^\pi(s, a) \\ \mathbb{E}_{a \sim \pi(s)}\left[A^\pi(s, a)\right] &= 0 \end{align}$$

实际计算：

$$Q(s, a) = V(s) + \left(A(s, a) - \frac{1}{|\mathcal{A}|}\sum_{a' \in \mathcal{A}} A(s, a')\right)$$ $V$

和 网络共享前面的卷积层。

$A$

30
31        # $V(s)$ — 状态价值
32        state_value = self.state_value(features)

$\varepsilon$-贪心探索策略

$$a = \begin{cases} \underset{a}{\arg\max} Q(s, a) & \text{概率 } 1-\varepsilon \\ \text{随机动作} & \text{概率 } \varepsilon \end{cases}$$ $\varepsilon$

随时间衰减：

$$\varepsilon_t = \varepsilon_{end} + (\varepsilon_{start} - \varepsilon_{end}) \times e^{-t/\text{decay}}$$

根据$\varepsilon$-贪心策略选择动作。

参数： - q_values: $Q(s, a;\theta)$ — 当前状态的Q值

返回值： - 选择的动作

3    def __init__(

16
17    def select_action(self, q_values: torch.Tensor) -&gt; int:
18        self.count += 1
19        # 衰减$\varepsilon$
20        self.epsilon = self.epsilon_end + (self.epsilon_start - self.epsilon_end) * (
21            -self.count / self.decay
22        )
23        self.epsilon = max(self.epsilon, self.epsilon_end)
24
25        if torch.rand(1).item() &lt; self.epsilon:
